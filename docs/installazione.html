Installazione rapida

Se hai Node installato, il comando seguente rileva gli agenti compatibili presenti sul computer.

npx skills add hypnosdesign/claude-skill-scrittura-italiana

Controlla sempre la destinazione proposta dal comando prima di confermare.

Claude Code

Disponibile in tutti i progetti

git clone https://github.com/hypnosdesign/claude-skill-scrittura-italiana ~/.claude/skills/scrittura-italiana

Disponibile in un solo progetto

Clona la repository in .claude/skills/scrittura-italiana nella radice del progetto. Riapri Claude Code dopo l'installazione.

Come si attiva

La descrizione della skill intercetta richieste di scrittura, correzione, revisione, argomentazione e dubbi sulla lingua italiana. Puoi anche nominarla esplicitamente.

Claude Desktop e claude.ai

  1. Scarica lo ZIP dalla sezione Releases o dalla voce Code, Download ZIP su GitHub.
  2. Apri Impostazioni, Capabilities, Skills.
  3. Carica la cartella o il file ZIP.

La disponibilità delle Skill e dell'esecuzione di codice dipende dal client e dal piano. Se la voce non compare, consulta la documentazione aggiornata del servizio.

Altri assistenti

Gli agenti compatibili con Agent Skills possono usare lo stesso comando npx skills add. Per Custom GPT, Gem e assistenti che accettano istruzioni o file di conoscenza, usa il documento completo.

Apri la versione in un solo file

Limite della versione singola

Il contenuto resta disponibile, ma si perde l'attivazione automatica e il caricamento progressivo dei riferimenti. Il prompt completo è inoltre più pesante.

Come usarla

Indica l'obiettivo e, quando conta, il livello d'intervento. Incolla poi il testo completo e specifica eventuali vincoli.

Correzione minima

Esegui un proofread. Correggi soltanto errori grammaticali e di punteggiatura. Non modificare lessico o struttura.

Revisione della tesi

Esegui un line edit conservativo di questo paragrafo. Migliora chiarezza e ritmo, ma conserva tesi, negazioni, grado di certezza e terminologia.

Riscrittura profonda

Riscrivi il testo in modo più naturale. Non aggiungere dati o esempi. Segnala separatamente le informazioni concrete che mancano.

Con un campione di voce

Usa il secondo testo come campione del mio stile. Revisiona il primo senza copiarne le frasi e senza aggiungere opinioni.

Per testi lunghi, lavora per capitoli e conserva un glossario di nomi, concetti e scelte terminologiche.

Aggiornare la skill

Se l'hai clonata con Git, entra nella cartella della skill ed esegui:

git pull --ff-only

Prima di aggiornare una copia modificata localmente, controlla git status e conserva le personalizzazioni. Le note di rilascio sono nel CHANGELOG.
